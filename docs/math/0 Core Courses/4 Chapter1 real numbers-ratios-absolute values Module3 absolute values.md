# Module 1-03 Absolute Value

## 1.Definition
- The absolute value of a positive number is __the number itself__;
  the absolute value of a negative number is __its opposite__;  
  the absolute value of zero is __zero__.  
  $
  \begin{array}{ll}
  \left|a\right|=
  \begin{cases}
  a & a\ge0 \\
  -a & a\lt0 \\
  \end{cases}
  \end{array}
  $

## 2.Focus 1
__The definition of absolute value__  
- According to the definition of absolute value,
  it only changes __the sign of negative numbers__,
  while it has __no effect on positive numbers or zero__.

### $\fbox{1}\:$Which of the following statements is correct $\small \mathpunct{?}$.
::: wrap  
$
\begin{array}{lllll}
\textsf{(A)} \: \textsf{the absolute value of a rational number is always greater than zero} & \\
\textsf{(B)} \: \textsf{the absolute value of a natural number equals its opposite} & \\
\textsf{(C)} \: \textsf{the absolute value of a prime number equals its opposite} & \\
\textsf{(D)} \: \textsf{the absolute value of an odd number equals itself} & \\
\textsf{(E)} \: \textsf{the absolute value of an irrational number is always greater than zero} & \\
\end{array}
$  
:::
#### Solution
::: wrap  
$
\begin{array}{ll}
\textsf{(A)} \: 0\in\mathbb{Q} \,❌\;\\
\textsf{(B)} \: \mathbb{N}\ge0 \Longrightarrow \left|a\right|=a \,❌\;\\
\textsf{(C)} \: \mathbb{P}\gt1 \Longrightarrow \left|a\right|=a \,❌\;\\
\textsf{(D)} \: \mathbb{Z_{\textsf{odd}}}<0 \Longrightarrow \left|a\right|=-a \,❌\;\\
\textsf{(E)} \: 0\notin\left\{\mathbb{R}\setminus\mathbb{Q}\right\} \,✅\;\\
\end{array}
$  
:::
#### Conclusion
- Derived Solution  
  ::: wrap
  > $\boldsymbol{(E)}$  
  > According to the Solution, get $\textsf{(E)}$, so choose $E$.
