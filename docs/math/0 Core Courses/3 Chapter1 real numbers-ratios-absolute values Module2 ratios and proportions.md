## 16.Fractions
- When the unit "$1$" is divided into __several equal parts__, a number that represents __one or several of these parts__ is called __a fraction__.  
  $
  \begin{array}{ll}
  \textsf{fractions}
  \begin{cases}
  \textsf{numerator}\lt\textsf{denominator (e.g. }\frac{3}{7}\textsf{)} & \textsf{proper fraction} \\
  \textsf{numerator}\ge\textsf{denominator (e.g. }\frac{7}{5}\textsf{)} & \textsf{improper fraction} \\
  \end{cases} \\
  \end{array}
  $  

## 17.Decimals
- __Integer part__
  - __Pure decimal__: a decimal whose integer part is $0$, e.g. $0.21$
  - __Mixed decimal__: a decimal whose integer part is not $0$, e.g. $3.21$
- __Decimal part__
  - __Terminating decimal__: e.g. 0.21
  - __Non-terminating decimal__
    - __Repeating decimal__
      - __Pure repeating decimal__: e.g. $0.212121\cdots$
      - __Mixed repeating decimal__: e.g. $0.3121212\cdots$
    - __Non-repeating decimal__: e.g. $\sqrt{2}, \pi, \mathrm{e}$

## 18.Conversion Between Decimals and Fractions
- Terminating decimals to fractions conversion
  - Use __$\boldsymbol{10, 100, 1000, \cdots}$ as the denominator__.
  - __Example__: $0.21=\frac{21}{100}$
- Pure repeating decimals to fractions conversion
  - Use __$\boldsymbol{9, 99, 999, \cdots}$ as the denominator__,
    where __the number of $\boldsymbol{9\textsf{'s}}$__ equals __the number of repeating digits__.
  - __The numerator is the repeating part__.
  - __Example__: $0.\dot{2}\dot{1}2121\cdots=\frac{21}{99}=\frac{7}{33}$
  - Formula: $0.\underbrace{\dot{a}b\cdots\dot{g}}_{\textsf{repeating digits}}=\frac{ab\cdots g}{\underbrace{99\cdots9}_{9\textsf{'s as repeating digits}}}$
- Mixed repeating decimals to fractions conversion
  - __The denominator consists of $\boldsymbol{9}\textsf{'s}$ and $\boldsymbol{0}\textsf{'s}$__
    - __the number of $\boldsymbol{9}\textsf{'s}$__ equals __the number of repeating digits__.
    - __the number of $\boldsymbol{0}\textsf{'s}$__ equals __the number of non-repeating digits__.
  - __The numerator__ is obtained by __subtracting the non-repeating part__
    from __the combined number of the non-repeating and repeating parts__.
  - __Example__: $0.3\dot{1}\dot{2}1212\cdots=\frac{312-3}{990}=\frac{309}{990}=\frac{103}{330}$
  - Formula: $0.\underbrace{ab\cdots c}_{m}\underbrace{\dot{d}\cdots\dot{g}}_{n}=\frac{ab\cdots g-ab\cdots c}{\underbrace{99\cdots9}_{m}\underbrace{00\cdots0}_{n}}$
    - $m$: __number of non-repeating digits__ after the decimal point.
    - $n$: __number of repeating digits__.

## 19.Percentages
- A percentage expresses __how many hundredths one number is of another__,
  and is usually represented by __the symbol "$\boldsymbol{\%}$"__.
